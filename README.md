# Smart Bharat AI 🚀

Smart Bharat AI is an all-in-one civic companion platform designed to streamline citizen services, simplify grievance redressal, and make government schemes easily accessible to everyone.

## 🌟 Key Features
- **AI-Powered Chatbot**: Get instant answers to your queries regarding government services.
- **Complaint Registration**: A seamless portal to report civic issues and track their status.
- **Document Assistant**: Know exactly which documents you need for various government services.
- **Government Scheme Recommender**: Personalized recommendations based on user profiles to ensure no one misses out on benefits.

## 🚀 How to Use
1. Clone this repository to your local machine.
2. Open `frontend/index.html` in your browser to explore the dashboard.
3. Use the **Ask AI** feature to interact with the chatbot.
4. Navigate to **Report Complaint** or **Document Assistant** to use specific utilities.

## 📝 Testing Guide
- **Chatbot**: Try asking "What is the status of my complaint?" or "How to apply for a passport?"
- **Scheme Recommender**: Enter age and gender to see personalized scheme suggestions.
- **Accessibility**: This platform is designed to be responsive and easy to navigate across all devices.
